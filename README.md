## Process

Install npm packages ⬇

```shell
npm install
```

Run application in develop mode 🏎️💨

> [!CAUTION]
> Currently only optimized for deployment!

```shell
npm run watch
```

Run application in production mode 🏎️💨

```shell
npm run deploy
```

Clean artifacts 🧹🪣

> [!TIP]
> Keep your things clean!

```shell
npm run clean
```
